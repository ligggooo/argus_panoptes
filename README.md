# jiliang_monitor

*集成组后台运维管理系统*

## v1.1.1 ---------------------------------
### 变更
1. 修改数据库
    sqlite >>>>>>>>  postgres
    "sqlite:///my_db.sqlite" >>>>>>>>>> "postgresql://postgres:123456@10.130.160.114:60030/my_test"
2. 增加服务规范
    后台在部署服务和删除服务的时候会执行install_{包名}.sh 和 remove_{包名}.sh来安装和卸载服务
    每一个服务包中都必须包含有这两个文件。
3. 同一个容器中，禁止不同软件版本的同一个服务出现
    后端新增逻辑处理
